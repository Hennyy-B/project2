# Commercalify 

## Overview

Commercalify is a modern, responsive web platform designed to showcase our company job openings and products we sell

## Pages

### 1. **Home** (`index.html`)

- Landing page with company introduction
- Product/service showcase
- Search functionality
- Quick navigation to other sections

### 2. **Jobs** (`jobs.html`)

- Browse all available job openings
- Detailed job descriptions including:
  - Salary ranges
  - Key responsibilities
  - Essential & preferable requirements
  - Reference numbers for tracking

### 3. **Apply** (`apply.html`)

- Application form for interested candidates
- Structured process for submitting applications
- Required fields for candidate information

### 4. **About** (`about.html`)

- Company information
- Team details
- Company values and mission
- Grid-based layout showcasing company info
